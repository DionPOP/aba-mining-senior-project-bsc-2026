Semantics Guide

The core of argumentation theory is to determine which sets of arguments, called extensions, can be considered "collectively acceptable". These extensions are built upon the foundational concepts of conflict-freeness and admissibility.

Conflict-Free Set: A set of arguments is conflict-free if no argument in the set attacks another argument within the same set.

Admissible Set: A conflict-free set is admissible if it can defend all of its members against any external attacks by using other members within the set.

Based on these principles, primary semantics are introduced as follows:

  • Complete Extension: An admissible set that contains every argument that it defends.
  • Grounded Extension: The smallest and unique complete extension, computed based on un-attacked arguments. It represents the most skeptical and undeniable viewpoint.
  • Preferred Extension: A maximal (largest) admissible set that represents a maximally coherent viewpoint.
  • Stable Extension: A conflict-free set that attacks every argument not contained within the set, representing a fully opinionated viewpoint.
  • Evaluation Strategy: Credulous accepts arguments in at least one extension. Skeptical accepts arguments only if they appear in all extensions.

Accepted Assumptions

Credulous uses the union of extension sets; skeptical uses the intersection. The selected claim is accepted only if it is derived from these assumptions through the ABA rules.
